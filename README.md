**Project 1:        
	TO-DO List Application (Python)**
---



### **Description** 



The TO-DO List Application is a simple command-line program developed in Python to help users manage their daily tasks.



The application provides the following features:



* Add new tasks to the task list.
* View all saved tasks in a numbered format.
* Display a message when no tasks are available.
* Exit the application safely.
* Handle invalid menu choices by prompting the user to try again.



The program uses a Python list to store tasks and a loop-driven menu system to allow continuous interaction until the user chooses to exit.



### How to Run (Commands to Start the Project)



**Step 1: Save the file**

Save the code in a Python file, for example: (todo\_list.py)



**Step 2: Open Terminal / Command Prompt**

Navigate to the folder where the file is saved: (cd path/to/project)



**Step 3: Run the program**

python todo\_list.py or, if your system uses Python 3 separately: (python3 todo\_list.py)



**Step 4: Use the menu**

The program will display:



==== TO-DO LIST MENU ====

1\. Add Task

2\. View Tasks

3\. Exit



Enter:



1 to add a task.

2 to view all tasks.

3 to exit the program.
